Projets Personnels Encadrés

Au cours des deux ans de BTS, deux projets ont été mis en place.
Nous avons choisi de développer un site web et une application de gestion pour l'entreprise fictive Apic, dont l'activité principale consiste à proposer des formations et des certifications apicoles.
Ces projets ont été développés à l'école, l'activité au sein de l'entreprise ne permettant pas de développement informatique.

Le contexte des projets est consultable ici

SP1 - Site Apic-web

Le site internet permet à un internaute de créer un compte client, qui lui permettra de choisir une formation apicole. L'utilisateur peut choisir sa fonction, particulier ou entreprise, et s'inscrire au formations ou certifications proposées. Il peut aussi à tout moment, modifier, consulter ou mettre a jour ses choix via son compte client.

Lien vers le site :
Apic Web

Pour ce projet, nous avons utilisé les langages PHP avec le framework Bootstrap, et SQL. Une base de données commune au site internet et à l'application sont hebergés sur 1and1.
Le projet a été piloté avec Gantt project, afin que chaque collaborateur ait un rôle sur le developpement du site et pour que le projet ait une durée fixe.
Nous avons utilisé Git comme gestionnaire de version du site.



SP2 - Logiciel de Gestion pour Apic Web

L'application aura pour but d'automatiser la gestion des données utilisateurs d'Apic web.
Cette application n'est utilisable que par le personnel de Apic.
